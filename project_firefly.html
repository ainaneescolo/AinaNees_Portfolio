Firefly

  • Gameplay Programmer
  • 2
  • 4 Weeks
  • Unity Engine

Firefly is a 2D platform game in horizontal orientation. It is a game of point A to point B. To make a more interesting product we focus on improving it visually with animations and effects and adding elements like the different endings to provide an element of replayability. In the game there is a local game save system, which saves both your number of lives and position as plants and enemies that you have already purified. There are different mechanics and behaviors of enemies, in addition to an extensive level design.
